O SISTEMA SE BASEIA DA ARQUITETURA CLIENTE SERVIDOR, ONDE O 
CLIENTE DESEJA CALCULAR UMA EXPRESSÃO E O SERVIDOR PROVÊ ESSE 
SERVIÇO. 

PARA ISSO O CLIENTE DEVE-SE CONECTAR A UM SERVIDOR 
ATIVO E ESSE POR SUA VEZ SOLICITA UMA THREAD PARA ATENDÊ-LO.
A THREAD UTILIZA UM OBJETO DO TIPO CALCULATOR PARA COLOCAR A 
EXPRESSAO NA NOTAÇÃO POS-FIXA, CALCULAR E RETORNAR O RESULTADO.

UTILIZAMOS UM CONJUNTO DE CLASSES IMPLEMENTADAS ANTERIOMENTE 
PARA REALIZAR O CÁLCULO.

PARA ENTENDER MAIS SOBRE O PADRÃO DE PROJETO VISUALIZE O ARQUIVO
APRESENTACAO.PDF.

TEMOS UM POOL DE THREADS COM highWaterMark = 5. HIGHWATERMARK 
"MARCA DA ÀGUA" FUNCIONA COMO O TAMANHO MÁXIMO DE OBJETOS A SEREM 
INSTANCIADOS E ALOCADOS NO POOL DE OBJETOS LIVRES.
CINCO THREADS SÃO INSTANCIADAS QUANDO O SERVIDOR É EXECUTADO
ASSIM QUANDO O CLIENTE SOLICITA NÃO HÁ CUSTO COM A CRIAÇÃO DE 
UMA THREAD, SERVER OBJETC POOL VERIFICA A PRÓXIMA THREAD LIVRE 
ALOCADA NA POOL DE OBJETOS LIVRES, PASSA ESSA THREAD PARA O POOL 
DE OBJETOS USADOS E RETORNA ESSA THREAD PARA USO DO CLIENTE.

PARA EXECUTAR PROJETO: 

1- EXECUTE A CLASSE SERVERM
2- EXCUTE A INTERFACE GRÁFICA TELA.JAVA
3- INFORME IP E PORTA DO SERVIDOR PARA CONECTAR
4- INFORME A EXPRESSAO E PRESSIONE CALCULAR
